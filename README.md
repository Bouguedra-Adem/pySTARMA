requires python 2.7

pySTARMA is a Python library for modelling space-time-ARMA processes. 
It follows the STARIMA model proposed by Pfeifer und Deutsch (1980).
It implements the estimation of the space-time autocorrelation function and the partial space-time autocorrelation function. It also implments the model estimation by kalman filtering, based on Cipra and Motykov√° (1987) and Cheysson (2016).


All documentation is in the "docs" directory
